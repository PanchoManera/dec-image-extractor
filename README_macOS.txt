RT-11 Extractor para macOS universal

Contenido del paquete:
- RT11Extract.app (Aplicacion GUI bundle completa)
- rt11extract_cli (CLI extractor universal)
- imd2raw (Conversor ImageDisk a DSK/RAW)

Uso:
- Hacer doble clic en RT11Extract.app para abrir la interfaz grafica
- Desde Terminal:
  ./rt11extract_cli imagen.dsk -o carpeta/ (extractor)
  ./imd2raw imagen.imd imagen.dsk (conversor)

Todos los ejecutables incluyen el backend completo integrado.
No necesitan archivos adicionales para funcionar.
